Short API reference

'/phone_data_service/phone_data' POST (create new item), GET[LIST]
'/phone_data_service/not_sent_phone_data' GET[LIST] отримати дані тільки для sent_times == 0.
'/phone_data_service/top_down_sent_phone_data' GET[LIST] Отримати дані тільки для sent_time != 0. Посортовано у спадаючому порядку за sent_times.
'/phone_data_service/phone_data/<int:id>' GET, PATCH, DELETE. Для вас достатньо використовувати GET - Отримати елемент за id.
'/phone_data_service/increment_sent_statistics/<int:id>' GET. Запит збільшує sent_times параметр на один. Повертає JSON цього рядка.


JSON params to PATCH:
['sent_times', 'phone_number', 'address_city', 'address_street', 'address_house', 'address_entrance', 'location_latitude', 'location_longitude', 'first_name', 'app']

Для GET[LIST] можна вказувати параметр ?page=3 після URL. Вертає дані відповідно до сторінки. Розберетеся.
